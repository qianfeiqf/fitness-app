# 健身训练小程序 — 视觉设计文档

> 本文档供视觉 AI 快速理解当前页面结构与设计风格，以便重新出一版视觉方案。

---

## 一、产品定位

面向追求质感的健身爱好者的训练记录工具，核心功能：计划管理、跟练记录、数据统计、个人目标追踪。

## 二、导航结构

底部 Tab Bar 4 个标签页：

```
┌─────────────────────────────────────┐
│           微信原生 Tab Bar           │
│  [训练]  [计划]  [统计]  [我的]      │
└─────────────────────────────────────┘
```

另有 2 个非 Tab 页面通过路由跳转进入：
- `onboarding` — 新用户引导（4 步向导）
- `training` — 训练中（沉浸式全屏）
- `complete` — 训练完成总结
- `history` / `history/detail` — 训练历史
- `plans/detail` / `plans/create` — 计划详情/创建
- `exercises/library` — 动作库

## 三、当前设计风格

### 风格关键词
**纯黑白极简 + 新拟物化 (Neumorphism)**

### 配色

| 用途 | 色值 | 说明 |
|------|------|------|
| 页面背景 | `#FFFFFF` | 纯白 |
| 主文字/主操作 | `#1A1A1A` | 近黑 |
| 辅助文字 | `#666666` | 中灰 |
| 次要文字 | `#999999` | 浅灰 |
| 占位文字 | `#B0B0B0` | 更浅灰 |
| 浅色区域背景 | `#F5F5F5` | 微灰 |
| 禁用/分割 | `#E0E0E0` / `#E5E5E5` | 极浅灰 |
| 破坏性操作 | `#8B0000` | 暗红（仅删除/放弃） |

**零彩色** — 无绿色成功、无红色警告、无蓝色信息。所有功能状态通过灰度 + 字重 + 图标区分。

### 阴影系统（新拟物核心）

三层阴影实现物理质感，组件与背景同色，靠光影制造浮雕：

| 名称 | 用途 | 值 |
|------|------|------|
| raised | 卡片凸起态 | `6rpx 6rpx 12rpx rgba(0,0,0,0.12), -6rpx -6rpx 12rpx rgba(255,255,255,0.8)` |
| pressed | 按下/选中内凹态 | `inset 4rpx 4rpx 8rpx rgba(0,0,0,0.12), inset -4rpx -4rpx 8rpx rgba(255,255,255,0.8)` |
| floating | 弹窗浮层 | `10rpx 10rpx 20rpx rgba(0,0,0,0.15), -10rpx -10rpx 20rpx rgba(255,255,255,0.9)` |

**交互反馈**：点击时 raised → pressed（阴影从外凸变内凹），无颜色变化。

### 字体

- 正文/标题：`system-ui, -apple-system, sans-serif`
- 数字（重量/次数/组数）：`'DIN Alternate', system-ui, sans-serif` + `font-variant-numeric: tabular-nums`

### 圆角

| 级别 | 值 | 用途 |
|------|------|------|
| 卡片 | 24rpx | 卡片、弹窗 |
| 输入框 | 16rpx | 输入框、小卡片 |
| 胶囊 | 9999rpx | 按钮、标签、进度条 |

### 间距

8rpx / 16rpx / 24rpx / 32rpx / 48rpx / 64rpx 六级。

### 边框

**全部移除**，用阴影 + 留白分隔模块。

## 四、页面详细描述

### 1. 引导页 `onboarding`

**布局**：全屏纵向，无系统导航栏。顶部进度点 → 中间内容区 → 底部按钮。

```
┌──────────────────────────┐
│  ● ● ○ ○        第 2/4 步 │  ← 进度点 + 步数
│                          │
│  每周训练几天？            │  ← 标题（粗体大字）
│  选择适合你的频率          │  ← 副标题（灰色）
│                          │
│  ┌──┐ ┌──┐ ┌──┐         │
│  │ 2│ │ 3│ │ 4│  ...    │  ← 天数卡片（凸起阴影）
│  │天│ │天│ │天│         │     选中态 → 内凹阴影
│  └──┘ └──┘ └──┘         │
│                          │
│        上一步   [下一步 →] │  ← 底部按钮
└──────────────────────────┘
```

4 个步骤：目标选择（2×2 网格卡片）→ 训练天数（数字卡片）→ 经验水平（纵向卡片列表）→ 推荐计划（纵向卡片列表 + 标签）。

```
┌──────────────────────────┐
│  组间休息         1:24    │  ← DIN 大字计时
```

**热身阶段**（替代主训练区）：热身组清单（序号圆 + 重量×次数 + 标签），底部主按钮 + 跳过链接。

## 五、关键 UI 模式总结

| 模式 | 描述 |
|------|------|
| **凸起卡片** | 白底 + raised 阴影，无边框。几乎所有内容容器 |
| **内凹输入** | 白底 + pressed 阴影。输入框、拨盘容器、进度条轨道 |
| **主按钮** | `#1A1A1A` 背景 + 白色文字 + raised 阴影。`:active` → pressed |
| **次要按钮** | `#F5F5F5` 背景 + 黑色文字 + raised 阴影 |
| **文字按钮** | 透明背景 + `#666666` 文字，无阴影 |
| **危险操作** | `#8B0000` 暗红文字/图标，仅用于删除和放弃 |
| **选中态** | 从 raised → pressed（阴影方向反转），无颜色变化 |
| **数字展示** | DIN Alternate 字体 + `tabular-nums`，粗体大号 |
| **标签/徽章** | `#F0F0F0` 背景 + 黑色文字，胶囊形 |
| **进度条** | 内凹轨道 + `#1A1A1A` 填充 |
| **列表分隔** | 无边框，靠阴影 + 间距分隔 |
| **弹窗** | 浮层阴影，居中或底部弹出，拖拽手柄 |
| **骨架屏** | 渐变动画 shimmer，`#F0F0F0` → `#E0E0E0` |
