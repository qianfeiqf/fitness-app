# 健身训练计划小程序

专为硬核健身者设计的微信小程序，核心功能：计划制定 → 训练执行 → 数据复盘 → 动态演进

## 技术栈

| 组件 | 技术选型 |
|------|----------|
| 小程序端 | 微信原生框架 (WXML + WXSS + JS) |
| 本地数据库 | LokiJS + IndexedDB |
| 后端部署 | 微信云托管 (容器化Node.js) |
| 数据库 | 云托管MySQL |

## 快速开始

### 1. 微信开发者工具配置

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具 → 新建项目 → 选择本项目目录
3. AppID 使用你的小程序AppID（可在微信公众平台获取）
4. 勾选"不使用云开发"（初期），后续接入云托管
